Fanen is pivoting to BINAH (Hebrew for 'Understanding'). PRD completed at docs/00-pm/binah.prd.md on 2026-03-25.

**Why:** The original Fanen targeted info-disadvantaged investors with a subscription model. BINAH shifts to a fully free model targeting passive-income/retirement-planning users with three differentiating features: BINAH Map (geopolitical visualization), Value Chain Tier 1~3 analysis, and a passive income goal calculator.

**How to apply:**
- Character: Bandi (firefly) replaces Finny (FinAI)
- Colors: Teal #0D9488 + Lime #A3E635 + Deep Navy #0F1923
- Revenue: Free-first, then CPA + B2B API
- Key new features: BINAH Map, Value Chain, Passive Income Hub, Bandi Chart Commentary, Morning Light
- Sprint 10 starts with brand rebranding, Sprint 11-12 for core differentiators
- Existing Sprint 1-9 codebase to be rebranded and reused
